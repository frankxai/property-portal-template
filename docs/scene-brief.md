# Scene Brief

## Surface

Property Intelligence OS portal template.

## Audience

Small landlords, property owners, renters, and prospects. First pilot is German-first and brother-first.

## First Read

The first viewport must show that this is an actual operating portal, not a generic landing page:

- public property evidence
- owner approval gates
- listing status
- support/renter portal paths

## Visual Direction

FrankX posture: practical, premium, direct, operational. Use obsidian base, teal system signal, amber approval warnings, strong typography, and real property media. Avoid generic purple SaaS gradients and decorative dashboard filler.

## Motion

V1 uses static composition and browser-native interactions only. No Track B scroll motion until the core portal workflow is validated.

## Asset Notes

The sample uses an Unsplash property image for template demonstration. Production installs must replace it with owned or approved property photography and update asset source notes.
