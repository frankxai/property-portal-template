# Portal Taste Standard

The portal should feel like a premium property operations desk.

## Visual

- property media leads
- status labels are visible
- spacing is calm
- cards only frame real tools or repeated items
- no generic abstract hero
- no fake luxury language

## Copy

- practical and direct
- approved facts only
- missing facts are named
- owner review is explicit
- urgent commitments are human-owned

## Mobile

- no horizontal scroll
- buttons fit
- first property action is reachable
- support path is clear
- text does not overlap media or cards

## Premium Definition

Premium here means lower uncertainty, clearer service, better media, faster answers, and visible owner discipline.
