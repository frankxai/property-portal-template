# Support

## Community Template

The free template includes the Next.js app, sample property data, runtime contracts, validation scripts, and production-hardening docs.

## Partner Implementations

Partners can sell:

- property content intake and photography setup
- Vercel deployment and domain setup
- database, auth, storage, and email wiring
- owner dashboard customization
- listing channel workflow setup
- renter knowledge base migration
- weekly managed operations

## Boundaries

This template is not legal, tax, tenant-screening, property-management, or emergency-response software. Production operators own compliance, renter communication, maintenance escalation, and data processing agreements.
